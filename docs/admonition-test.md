# Custom Genestack Admonition

There is now a 'genestack' admonition type:

```
!!! genestack
    [Genestack](https://github.com/rackerlabs/genestack){:target="_blank"}
    is a cool new project!
```

It renders like this:

!!! genestack
    [Genestack](https://github.com/rackerlabs/genestack){:target="_blank"}
    is a cool new project!

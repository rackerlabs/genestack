### Skyline Maintenance: 2024.1 to 2025.1 Helm Migration

### Notes

# (opt) refers to /opt, the standard install path for Genestack.
# (etc) refers to /etc, the standard override path for Genestack.
# Replace all placeholder values in angle brackets before using this runbook.
# Keep this runbook operational. Prefer exact commands, exact file paths, expected outputs, and explicit validation steps.

### Validation

# Kubernetes 1.33.x satisfies the published OpenStack Helm Skyline chart minimum.
# Upgrading from Skyline 2024.1 (kustomize) to 2025.1 (Helm) requires uninstalling the kustomize deployment and installing the Helm chart.
# The existing skyline-apiserver-secrets Kubernetes secret is compatible with the Helm chart and does not need to be recreated.
# This migration removes the kustomize-based deployment and replaces it with the upstream OpenStack Helm chart.

### Goal

# Migrate Skyline from kustomize 2024.1 to Helm 2025.1 without service regression and with all workloads returning healthy.

## Apply Required Overrides or Patches

  skyline:

## Uninstall the Existing Kustomize Deployment

kubectl --namespace openstack delete -k /opt/genestack/base-kustomize/skyline/base

# Verify deletion:
kubectl --namespace openstack get pods -l application=skyline
kubectl --namespace openstack get svc skyline-apiserver

# Expected:
# - No Skyline pods remain
# - The skyline-apiserver service is deleted

## Create the Skyline Database Password Secret

# The Helm chart requires a new secret named skyline-db-password with the key db-password
# This is separate from the existing skyline-apiserver-secrets secret

# Generate a secure random password:
export DB_PASSWORD=$(openssl rand -base64 32)

# Create the secret with the database password:
kubectl --namespace openstack create secret generic skyline-db-password \
  --from-literal=db-password="$DB_PASSWORD"

# If the secret already exists, update it instead:
kubectl --namespace openstack patch secret skyline-db-password --type=merge -p "{\"data\":{\"db-password\":\"$(echo -n "$DB_PASSWORD" | base64)\"}}"

# Verify the secret was created:
kubectl --namespace openstack get secret skyline-db-password -o yaml

# Expected:
# - The secret contains the db-password key with the base64-encoded password
